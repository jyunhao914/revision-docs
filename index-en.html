Tutorial

Five steps from PDF to perfect presentation

1

Import Files

Click + Files in the top-left to select PDF or image files. Supports selecting multiple files at once.

You can also right-click any PDF or image in Finder → "Open With" → ReVision to import directly.

💡 Tip: After importing, clicking + Files again appends to existing pages, combining multiple sources.
2

Auto Text Recognition

After import, ReVision automatically uses macOS native OCR to detect all text, highlighted with green masks.

Green areas will have text removed, backgrounds AI-repaired, then restored as editable text boxes.

Zoomed in — the green rectangles are the text areas detected by OCR:

⚠️ Warning: Text in tables and charts will also be detected. Use the "Erase" tool to remove masks from areas you want to keep.
3

Manually Adjust Masks

This is ReVision's core advantage — when AI detection isn't perfect, you can directly make manual corrections:

Mark
Select additional text areas to remove (add green mask)
Erase
Remove masks from areas that should be kept
+
Zoom in (or scroll up)
−
Zoom out (or scroll down)
Fit
Auto-fit to window size
Magnifier
Real-time magnification around cursor
Undo
Undo last operation
Reset
Return to initial auto-detection state
💡 Tip: Press E to quickly toggle between Mark and Erase modes.
4

Manage Pages

The left panel is your page management area:

  • Click thumbnail — switch to that page
  • Drag thumbnail — reorder pages
  • Right-click thumbnail — show menu (delete page)
  • + Files — append more PDFs or images
  • Close Files — clear all and start over
💡 Tip: Use ↑ ↓ or ← → arrow keys to quickly switch pages.
5

Export PPTX

Once satisfied with every page, click Export PPTX in the top-right. ReVision processes each page:

  • Render at 200 DPI high quality
  • AI smart background repair (auto-selects best strategy)
  • Precisely detect font size, color, bold
  • Generate editable text boxes
  • Output standard .pptx file (opens in PowerPoint / Keynote / Google Slides)
⚠️ Warning: Complex background (photo pages) AI repair takes longer (~10 sec/page). Please be patient.

⚠️ Notes & FAQ

1

Tables & Charts

OCR detects all text, including text within tables and charts. If you don't want to remove that text, use the "Erase" tool to remove the green mask from those areas, keeping them as part of the background.

2

High-Resolution Images

When importing ultra-high-resolution images (e.g., camera photos at 4000x6000), AI repair automatically downscales to 2048px for processing then restores, ensuring export doesn't hang. Non-masked areas retain original quality.

3

Export Time

Processing time per page depends on background complexity:

  • Solid background pages: ~2-3 seconds
  • Photo / complex background pages: ~10-15 seconds (LaMa AI inference)
  • You can click "Cancel" at any time during export
4

Supported Formats

Input supports PDF, PNG, JPEG, BMP. You can mix different formats and export as a unified PPTX.

PDFs automatically detect page dimensions (A4, 16:9, 4:3, etc.). Non-standard sizes are automatically adapted.

5

OCR Languages

Uses the macOS native Vision OCR engine, supporting Traditional Chinese, Simplified Chinese, English, Japanese, and many other languages. Recognition accuracy continues to improve with macOS updates.
